# 🚀 Mode Graphique - Démarrage Rapide

## ✅ Installation Terminée

Le mode graphique est maintenant intégré à votre calculatrice !

## 📊 Accès Rapide

1. Lancez l'application
2. Cliquez sur **📈 Graphique** dans la sidebar
3. Entrez une fonction
4. Cliquez sur **📊 Tracer le Graphique**

## ⚡ Exemples Rapides

### Fonctions de Base
```
x^2              → Parabole
x^3              → Cubique
sin(x)           → Sinus
cos(x)           → Cosinus
exp(x)           → Exponentielle
ln(x)            → Logarithme
sqrt(x)          → Racine carrée
abs(x)           → Valeur absolue
1/x              → Inverse
```

### Fonctions Composées
```
x * sin(x)       → Oscillations croissantes
sin(x) / x       → Fonction sinc
exp(-x^2)        → Gaussienne
x^3 - 3*x        → Cubique avec inflexion
1 / (1 + x^2)    → Lorentzienne
```

## 🎯 Utilisation

### Méthode 1 : Saisie Manuelle
1. Taper la fonction dans le champ
2. Exemple : `x^2 + 2*x - 1`
3. Cliquer sur "Tracer"

### Méthode 2 : Exemples Rapides
1. Cliquer sur un exemple (Parabole, Sinus, etc.)
2. Le graphique s'affiche automatiquement
3. Modifier si nécessaire

## 💡 Astuces

### Combiner des Fonctions
```javascript
sin(x) + cos(x)
x^2 * exp(-x)
sqrt(abs(x))
```

### Utiliser π et e
```javascript
π * x
e^x
sin(2*π*x)
```

## 🐛 Problèmes Courants

### "Fonction invalide"
- Vérifier l'orthographe : `sin`, `cos`, `tan`
- Utiliser `*` pour multiplier : `2*x` pas `2x`
- Utiliser `^` pour puissance : `x^2` pas `x²`

### Graphique ne s'affiche pas
```bash
# Vérifier que Chart.js est installé
cd calculator-react
npm list chart.js react-chartjs-2
```
